# Official and Reference MCP Server Repositories

Here are the links to the requested Model Context Protocol (MCP) server GitHub repositories:

## 1. Google Workspace (Official) MCP Server
* **Repository**: https://github.com/gemini-cli-extensions/workspace
* **Description**: Google's official Model Context Protocol (MCP) server for Google Workspace applications (Gmail, Drive, Calendar, Docs, Sheets, Slides, etc.).

## 2. Google Maps (Official) MCP Server
* **Repository**: https://github.com/gemini-cli-extensions/maps-grounding-lite
* **Description**: Google's offical MCP server for Google Maps. You need an API key. The free limit is enough for personal use, however you need to pay for high usage.

## 3. GitHub MCP Server (Official)
* **Repository**: https://github.com/github/github-mcp-server
* **Description**: The official, production-ready GitHub MCP server maintained by GitHub, written in Go. It enables AI tools and assistants to manage repositories, issues, pull requests, and other workflows directly.

## 4. Last 30 Days GitHub Repo
* **Repository**: https://github.com/mvanhorn/last30days-skill
* **Description**: An MCP-compatible skill (`last30days-skill`) designed to perform real-time, cross-platform research and synthesize citation-backed briefings using content specifically from the last 30 days.

## 5. Data Commons Google Official MCP Server
* **Repository**: https://github.com/gemini-cli-extensions/datacommons
* **Description**: The official Google's Data Commons MCP. You need a free API key from Data Commons, it's easy to get and set up.

## 6. Polymarket MCP Server
* **Repository**: https://github.com/caiovicentino/polymarket-mcp-server
* **Description**: Open-source MCP server enabling AI agents to fetch Polymarket prediction market details, current prices, and event odds.
